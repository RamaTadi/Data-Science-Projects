### Installation 
---
The project is done online on kaggle notebook. 
To to this one needs

* Kaggle Notebook or Jupyter Notebook
* Python 3.7.12
* Python packages
    * Pandas
    * Numpy
    * Matplotlib
    * Seaborn 
    * Sklearn 
    * Statsmodel
